How to Add Files

To add new files to the archive:

  1. Fork this repo
  2. Upload your file to the appropriate directory:
    • ipa/ for IPAs
    • deb/ for DEBs
    • dylib/ for DYLIBs
  3. Create a pull request
  4. Once merged, the file(s) will automatically appear on this site

Direct download link formats:

  • https://ipa.backupbot.net/ipa/{FILENAME}.ipa
  • https://ipa.backupbot.net/deb/{FILENAME}.deb
  • https://ipa.backupbot.net/dylib/{FILENAME}.dylib
